# 功能：玩家移動系統

## 描述
實作 2D 橫向捲軸遊戲的玩家角色移動系統，包含左右移動、跳躍、重力物理，以及攝影機跟隨。

## 需求

### 玩家角色
1. 在畫面中渲染玩家角色（初期使用簡單的矩形或圓形）
2. 玩家有位置 (x, y)、速度 (vx, vy)、尺寸等屬性
3. 玩家受重力影響

### 鍵盤控制
1. **A 或 ←**：向左移動
2. **D 或 →**：向右移動
3. **W 或 ↑ 或 空白鍵**：跳躍（僅在地面上時）
4. 平滑的移動體驗

### 物理系統
1. 重力：玩家持續受到向下的重力
2. 地面碰撞：玩家不會穿過地面
3. 跳躍：施加向上的初速度
4. 速度限制：防止過快移動

### 攝影機跟隨
1. 攝影機水平跟隨玩家
2. 平滑的跟隨效果（不要太僵硬）
3. 保持玩家在畫面中央附近

## 驗收標準
- [ ] 玩家角色在畫面中可見
- [ ] 使用 A/D 或方向鍵可以左右移動
- [ ] 按下跳躍鍵可以跳躍
- [ ] 玩家受重力影響並會落地
- [ ] 玩家不會穿過地面
- [ ] 攝影機會跟隨玩家移動
- [ ] 移動流暢，無明顯卡頓
